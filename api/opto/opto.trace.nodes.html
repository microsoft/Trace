Backward pass.

feedback: feedback given to the current node propagate: a function that takes in a node and a feedback, and returns a dict of {parent: parent_feedback}.

def propagate(node, feedback):

return {parent: propagated feedback for parent in node.parents}

visualize: if True, plot the graph using graphviz reverse_plot: if True, plot the graph in reverse order (from child to parent). print_limit: the maximum number of characters to print in the graph.
